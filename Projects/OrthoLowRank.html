Orthogonal Low Rank Embeddings for Image Classification

DenseNet, OLE Loss, Angular Softmax and Probability calibration

Abstract

This work uses additive softmax and calculates parameterized values of deep features based on theta, likely facilitating angular loss calculations within the model. DenseNet is used as the architecture of the model, complemented by orthogonal Low Rank Embedding (OLE) loss function trained on the CIFAR-10 dataset. This project handles the computation of OLE loss and its integration into the training process of the model in a novel way so as to ensure quick training. To address the issue related to classifcation model confidence estimation, this project shows multiple softmax calibration methods like temperature scaling, to refine the model's confidence estimates.
