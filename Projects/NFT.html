Minting NFTs

Deploying unique NFT collection on the Ethereum blockchain.

Summary

In this project, I developed and deployed my own NFT collection on the Ethereum blockchain, following a comprehensive process. I began by uploading NFT images to IPFS using services like Pinata or NFT Storage, then created metadata files in JSON format with links to the IPFS-stored images. The core of the project involved developing a Solidity smart contract using OpenZeppelin's ERC721 standard, which I customized with features like a token ID counter, maximum supply limit, and a base URI for metadata. I implemented a custom tokenURI function and added a secure withdraw function with access control. I compiled, tested, and deployed the contract to an Ethereum testnet (Sepolia), then verified it on Etherscan. Futher, I minted NFTs using the deployed contract's functions and verified their appearance on OpenSea Testnets. Key technical aspects included working with IPFS, implementing ERC721 standards, using Remix IDE for development and deployment, and integrating with OpenSea for NFT visualization and trading. NFT is deployed on a testnet, please do not make any offers to buy it :D
