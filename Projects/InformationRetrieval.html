Information Retrieval (Passage Ranking) using NLP

Microsoft India AI Challenge 2018

Abstract

This is a solution to Microsoft's AI Challenge 2018, focusing on passage retrieval from a given dataset using machine learning models. Given a user query and candidate passages corresponding to each, the task is to mark the most relevant passage which contains the answer to the user query. While the competition is no longer active for submissions, the solution remains accessible for educational purposes. The implementation comprises two main approaches: BM25 and Deep Learning (DL). The BM25 method, a well-known probabilistic technique for passage retrieval, is implemented with minor optimizations. Additionally, various deep learning models are explored for passage retrieval tasks, including Convolutional Neural Networks (CNN), Long Short-Term Memory (LSTM), and attention mechanisms with LSTM. Notably, a Bi-directional Attention Flow (BIDAF) model is adopted in this solution. Implementation details include hardware specifications, with training performed on an 8 GB GeForce 960M GPU and 8 GB RAM. Given resource constraints, better GPU resources are recommended for improved efficiency. TensorFlow libraries are utilized and the users are directed to the code for specific dependencies. Data acquisition is expected from the competition's link, and GloVe word embeddings are required, with instructions for obtaining 100-dimensional embeddings provided. The solution offers valuable insights into both traditional probabilistic methods and state-of-the-art deep learning approaches for passage retrieval, serving as a practical resource for understanding machine learning principles in real-world scenarios.
